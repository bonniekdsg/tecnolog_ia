Dúvidas Frequentes

Como solicito acesso ao Hub tecnolog.IA?

add
O acesso é restrito a membros e servidores do MPAC. A autenticação é feita via Single Sign-On (SSO) utilizando suas credenciais institucionais padrão.

As ferramentas utilizam dados sigilosos?

add
Sim, o tecnolog.IA foi projetado com segurança de nível governamental. O processamento de dados ocorre em ambiente controlado e as informações não são compartilhadas com modelos públicos externos.

Posso integrar o Anton.IA com meus processos?

add
A Anton.IA atua como o motor de integração central. Você pode utilizá-la para conectar diferentes fluxos de trabalho e bases de dados institucionais via API Gateway.

Existe limite de uso para transcrição de áudio?

add
O TranscreveAI possui uma cota generosa para garantir a disponibilidade. Para processamentos volumosos ou urgentes, entre em contato com o Seringal Lab.
